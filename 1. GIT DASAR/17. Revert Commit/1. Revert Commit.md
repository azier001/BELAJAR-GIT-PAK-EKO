**Revert commit** adalah fitur Git yang memungkinkan kita untuk **membatalkan commit** yang sudah kita lakukan sebelumnya. 

### ğŸ¯ Cara Kerja Revert

- Git **tidak menghapus** commit yang salah
- Melainkan **membuat commit baru** yang membatalkan perubahan dari commit sebelumnya
- Riwayat commit tetap aman dan terjaga

### ğŸ’¡ Contoh Kasus

Misalnya kita sudah melakukan commit untuk mengubah data dari text **"Dina"** menjadi **"Dini"**. 

Jika kita melakukan **revert**, maka Git akan:
- Secara otomatis membuat commit baru
- Commit baru ini akan mengubah kembali dari **"Dini"** ke **"Dina"**
- Commit lama tetap ada di history
