## ğŸ Apa yang Didapat Setelah Clone?

Setelah melakukan clone, kita otomatis mendapatkan:

### 1ï¸âƒ£ Remote Repository Origin

- Git akan otomatis menambahkan **remote** bernama `origin`
- `origin` menunjuk ke URL repository yang kita clone
- Ini memudahkan kita untuk push dan pull nantinya

**Cek remote dengan:**
```bash
git remote -v
```

**Output:**
```
origin  https://github.com/username/project-name.git (fetch)
origin  https://github.com/username/project-name.git (push)
```

### 2ï¸âƒ£ Branch Utama

- Clone akan otomatis men-download **branch utama** dari remote
- Biasanya bernama `main` atau `master`
- Kita langsung berada di branch ini setelah clone

**Cek branch dengan:**
```bash
git branch
```

**Output:**
```
* main
```
