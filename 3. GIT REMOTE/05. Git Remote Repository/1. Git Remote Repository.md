## â“ FAQ (Pertanyaan Umum)

### **Q: Apakah wajib menggunakan nama "origin"?**
**A:** Tidak wajib, tapi **sangat disarankan** karena ini adalah standar konvensi di Git.

### **Q: Bisa menambahkan lebih dari 1 remote?**
**A:** **Bisa!** Misalnya:
```bash
git remote add origin git@github.com:user/repo.git
git remote add backup git@gitlab.com:user/repo.git
```

### **Q: Apa bedanya SSH URL dengan HTTPS URL?**
**A:** 
- **SSH**: `git@github.com:user/repo.git` (lebih aman, butuh setup SSH key)
- **HTTPS**: `https://github.com/user/repo.git` (lebih mudah, butuh username/password)
